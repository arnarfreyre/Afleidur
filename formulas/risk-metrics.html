Risk Metrics

Quantitative risk measurement formulas

Expected Credit Loss (ECL)

$$ECL = EAD \times PD \times LGD$$

Variables:

  • $EAD$ = Exposure at Default
  • $PD$ = Probability of Default
  • $LGD$ = Loss Given Default

Value at Risk (VaR)

$$VaR_{\alpha} = -\inf\{x : P(L > x) \leq 1 - \alpha\}$$
Confidence levels: Typically 95% or 99%

All formulas verified using Python calculations
