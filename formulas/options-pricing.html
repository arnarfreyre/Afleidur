Options Pricing

Black-Scholes model and option Greeks

Black-Scholes Call Option

$$C = S_0 N(d_1) - Ke^{-rT} N(d_2)$$
$$d_1 = \frac{\ln(S_0/K) + (r + \sigma^2/2)T}{\sigma\sqrt{T}}$$
$$d_2 = d_1 - \sigma\sqrt{T}$$

Variables:

  • $C$ = Call option price
  • $S_0$ = Current stock price
  • $K$ = Strike price
  • $r$ = Risk-free rate
  • $T$ = Time to maturity
  • $\sigma$ = Volatility
  • $N(x)$ = Cumulative standard normal distribution

Black-Scholes Put Option

$$P = Ke^{-rT} N(-d_2) - S_0 N(-d_1)$$
Put-Call Parity: $C - P = S_0 - Ke^{-rT}$

Option Greeks

Delta (Δ)

$$\Delta_{call} = N(d_1), \quad \Delta_{put} = -N(-d_1)$$

Gamma (Γ)

$$\Gamma = \frac{\phi(d_1)}{S_0 \sigma \sqrt{T}}$$

Theta (Θ)

$$\Theta_{call} = -\frac{S_0 \phi(d_1) \sigma}{2\sqrt{T}} - rKe^{-rT}N(d_2)$$

Vega (ν)

$$\nu = S_0 \phi(d_1) \sqrt{T}$$

Rho (ρ)

$$\rho_{call} = KTe^{-rT}N(d_2), \quad \rho_{put} = -KTe^{-rT}N(-d_2)$$

All formulas verified using Python calculations
