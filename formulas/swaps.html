Swaps

Interest rate and currency swap valuation

Interest Rate Swap Valuation

$$V_{swap} = B_{fixed} - B_{floating}$$
Note: At inception, swap value = 0 (par swap rate)

Swap Rate

$$r_{swap} = \frac{1 - P(0,T_n)}{\sum_{i=1}^{n} P(0,T_i) \Delta t_i}$$

All formulas verified using Python calculations
