Time Value of Money

Fundamental formulas for financial calculations

Compound Interest (Discrete)

$$FV = PV \times (1 + r)^n$$

Variables:

  • $FV$ = Future Value
  • $PV$ = Present Value
  • $r$ = Interest rate per period
  • $n$ = Number of periods
Note: For finding periods: $n = \frac{\ln(FV/PV)}{\ln(1+r)}$

Present Value of Single Cash Flow

$$PV = \frac{CF}{(1 + r)^t}$$

Variables:

  • $CF$ = Cash flow at time $t$
  • $r$ = Discount rate
  • $t$ = Time period
Usage: Discounting future cash flows to present value

Annuity Present Value

$$PV = C \times \frac{1 - (1 + r)^{-n}}{r}$$

Variables:

  • $C$ = Regular payment amount
  • $r$ = Interest rate per period
  • $n$ = Number of payments
Application: Used for bonds with regular coupon payments

Annuity Future Value

$$FV = C \times \frac{(1 + r)^n - 1}{r}$$

Variables:

  • $C$ = Regular payment amount
  • $r$ = Interest rate per period
  • $n$ = Number of payments
Usage: Calculate future value of regular investments

Perpetuity Present Value

$$PV = \frac{C}{r}$$

Variables:

  • $C$ = Constant payment amount
  • $r$ = Interest rate per period
Note: Assumes payments continue forever

Growing Perpetuity

$$PV = \frac{C}{r - g}$$

Variables:

  • $C$ = Initial payment amount
  • $r$ = Discount rate
  • $g$ = Growth rate (must be less than $r$)
Application: Gordon Growth Model for stock valuation

Continuous Compounding

$$FV = PV \times e^{rt}$$

Variables:

  • $e$ = Euler's number (≈ 2.71828)
  • $r$ = Annual interest rate
  • $t$ = Time in years
Usage: Common in derivatives pricing models
